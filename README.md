Elixir/Erlang daily helpers and learning playground.

## Check ehelper archive in your project

```
# in mix.exs project/0

archives: [{:ehelper, "~> 0.2"}]
```

## Install

```bash
mix archive.install hex ehelper --force
mix h
mix local

## locally install or update
mix up
```
